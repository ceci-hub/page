Porque tomar este curso?

The simple graph has brought more information to the data analyst’s mind than any other device.— John Tukey

  • Las figuras y tablas cuentan una historia.
    • Es lo primero que un lector casual ve, ¡ tal vez lo único!
  • Nos permiten entender los datos y anticipar lo que nos van a decir los modelos.
    • Muchas veces, diagnosticar un mal modelo.

Un ejemplo cierre.

  • La Cuarteta de Ascombe: es un set de 4 datasets, donde tenemos pares \((x_i,y_i)\) continuos.
  • Es natural querer empezar a sacar estadísticos o correr regresiones.
  • Pero esto nos puede llevar a cometer errores.
    • Es un ejemplo Edward Tufte que nos muestra la importancia de mirar los datos antes de analizarlos.
    • Es una idea ‘vieja’ pero que es cada vez más importante.

Regresión.

  • Podemos correr una regresión pensando en un modelo lineal \(y_i=\beta_0+\beta_ix_i+e_i\)

Regresión.

Code

suppressPackageStartupMessages(library(tidyverse))
library(kableExtra)
library(xtable)

Q<-Tmisc::quartet #<< Levanto la data!
Q1<-Q %>% filter(set=="I")
Q2<-Q %>% filter(set=="II") 
Q3<-Q %>% filter(set=="III")
Q4<-Q %>% filter(set=="IV")

lm1<-lm(y~x,Q1)
lm2<-lm(y~x,Q2)
lm3<-lm(y~x,Q3)
lm4<-lm(y~x,Q4)

stargazer::stargazer(lm1, lm2, lm3,lm4, type = "html",omit.table.layout = "sn")
