Datos ordenados - Tidy.

  • Es un enfoque de tener los datos almacenados en un formato ordenado.

  • De acuerdo con este enfoque los datos están ordenados cuando.

    1. Cada fila es una observación.
    2. Cada columna es una variable.
    3. Cada celda tiene un solo valor.
  • Cuando tenemos datos desordenados?
  • Los datos nos llegan en formato super variados, un formato que puede ser útil para almacenar puede no ser el más útil para analizar.

Pivotear Largo

El problema:

  • Una herramientas fundamental para preparar bases son las funciones pivot.

  • La base de datos puede con variables en múltiples columnas, pero precisamos poner variables en las filas.

  • Pivot Longer nos permite ‘estirar para abajo’ la base, pasamos columnas a filas definiendo el dónde vamos a guardar el nombre de la variable y los valores.

Join

Joining data frames

Funcion General de R

something_join(x, y)

Funcion General de P

pd.merge(x,y,on=‘key’,how=‘something’)

Joining data frames

  • left_join(): El resultado es tener TODAS las observaciones del dataset L (izquierdo), pero solo las observaciones del dataset R (derecho) donde la clave (ID) se encuentran en L
  • right_join():El resultado es tener TODAS las observaciones del dataset R, pero solo las observaciones del dataset L donde la clave (ID) se encuentran en R
  • full_join(): El resultado contiene todas las observaciones de L y todas las de R, sin importar si tienen un matching key o no.
  • inner_join(): El resultado son las observaciones de L y R que tienen un matching key en el otro.
